# scholarup
A Chrome extension for collecting bibliographic material from Google Scholar.

1. Download this repository, unzip, and put it somewhere useful. 

2. Open Chrome and go to `chrome://extensions/`.

3. Turn on Developer mode (top right).

4. Click `Load unpacked.`

5. Select the scholarup folder.

6. Go to Google Scholar, either the results page for a search or the 'cited by' link for a given work.

7. Click the Puzzle piece icon in Chrome, find "Scholar Metadata Extractor," and click it to open the popup.

8. Click 'collect & go to next'. This will collect the first page of results. If the pop up disappears, just hit the extension icon again. You can page through collecting results; download as json; or clear the collector.

I mean, yes, zotero can do all this kind of thing. I just wanted to see how to do it. And maybe having this data as json is handy for you.
